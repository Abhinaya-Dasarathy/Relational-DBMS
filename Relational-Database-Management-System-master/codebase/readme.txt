Relational Database Management System Design and Implementation                    

• Designed an architecture and developed a Relational Database System in C++ from scratch.

• Developed relation manager API to manage tuple-level operations, B+ tree based index manager, and query engine to support the operators of query in a database system. 

Language: C++

There are 4 folders which has code related to the following-

rbf- Record-based file manager
rm- relation manager
ix- index component
qe- query engine

ALl the test cases in the folders have been verified.
